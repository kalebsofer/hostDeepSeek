# hostDeepSeek

## Overview
hostDeepSeek is a Dockerized implementation of the DeepSeek-R1-Distill-Qwen-7B language model, designed for self-hosting on remote servers. This project provides an easy-to-deploy solution for running the powerful 7B parameter model in a containerized environment, accessible via API endpoints.

## Features
- 🐳 Docker container for easy deployment
- 🤖 DeepSeek-R1-Distill-Qwen-7B model integration
- 🌐 API access for remote inference
- ⚙️ Optimized for server deployment
- 🔒 Self-hosted solution for data privacy

## Quick Start
1. Clone the repository
2. Build the Docker image
3. Configure environment variables
4. Run the container
5. Access the API endpoints

## API Usage
The hosted model provides RESTful API endpoints for:
- Text generation
- Chat completion
- Model inference

## Requirements
- Docker
- NVIDIA GPU (recommended)
- CUDA support
- Python 3.8+
